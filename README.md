# PD_Tool
A simple tool for working with Project Diva DT/FT/F/F2/X files

# Dependencies:
+ `.NET Framework 4.6.1`: Required to run/build PD_Tool C# project.

# Tools:
- `FARC Extract/Create`
- `DIVAFILE Encrypt/Decrypt`
- `DB_Tools`
  - `Aet DB Converter`
  - `Auth DB Converter`
  - `Spr DB Converter`
- `Converting Tools`
  - `A3DA Converter`
  - `DataBank Converter`
  - `DEX Converter`
  - `DIVA Converter`
  - `STR Converter`
  - `VAG Converter`
